---

Vibecode includes a growing catalog of integrations for AI models, media generation, APIs, communication, video, and agent experiences.

The easiest way to add one is to describe what you want in plain English:

- "Add GPT chat to my app"
- "Generate images from text"
- "Send transactional email"
- "Add weather data"
- "Build an in-app coding agent"

Vibecode will wire up the backend calls, environment variables, and project changes for you.

Browse the full catalog here:

- [Integrations Overview](/integrations/index)
- [Public integrations catalog](https://www.vibecodeapp.com/integrations)
