Vibecode comes with a library of built-in integrations — AI models, image generators, audio tools, data APIs, and more. All integrations are billed through your Vibecode credits at no markup. No separate accounts or API keys to manage.

## How Integrations Work

1. **Tell the AI what you need** — "Add AI chat to my app" or "Let users generate images"
2. **The AI sets it up** — It wires up the API call in your backend, keeping keys secure
3. **Usage is billed to your credits** — No external accounts, no extra configuration

All API calls go through your backend server, so keys are never exposed to users.

## Available Integration Categories

| Category | Examples |
|----------|----------|
| **Text Generation** | OpenAI GPT-5 Mini, Google Gemini, Grok 4 Fast |
| **Image Generation** | OpenAI GPT Image 1, Nano Banana, Ideogram 3.0 |
| **Audio** | ElevenLabs TTS, ElevenLabs Music, Sound Effects, Audio Transcription |
| **Data APIs** | Apple Maps, OpenWeather, TMDB, Stock & Crypto data |
| **Communication** | Resend Email, Expo Push Notifications |
| **Device Features** | Haptic Feedback, Runtime Logs |

<Tip>You can use multiple integrations in the same app. A recipe app could use GPT for generating recipes, image generation for food photos, and OpenWeather for seasonal suggestions.</Tip>

For detailed setup guides for each integration, see the [Integrations reference](/integrations/index).
