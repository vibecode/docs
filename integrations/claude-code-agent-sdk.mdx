Claude Code (Agent SDK) lets you add coding-agent style experiences directly into your app.

This is useful for products that need multi-step reasoning, tool use, task execution, or agent-style coding workflows for end users.

## Best For

- In-app coding agents
- Developer copilots
- Multi-step task execution
- Agentic workflow products

## Features

- Multi-step reasoning
- Tool use and action-taking
- Conversation management
- Strong fit for coding-agent experiences

## Ask Vibecode

Tell the AI:

> "Add the Claude Code Agent SDK to my app so users can chat with an agent that can reason, use tools, and complete coding tasks."

The AI will wire up the integration and backend flow for you.

## Choose This If

Choose Claude Code (Agent SDK) when you want an agent that can take actions and work across multiple steps. If you only need normal AI chat, a standard text model like [OpenAI GPT-5.1](/integrations/openai-gpt-5-1) or [OpenAI GPT-5 Mini](/integrations/openai-gpt-5-mini) is usually simpler.
