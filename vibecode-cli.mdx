---

Vibecode CLI connects your AI agent to Vibecode infrastructure so it can build and deploy apps with the same core platform capabilities available inside Vibecode.

It is designed for agent-first workflows and works with tools like Cursor, Claude, Codex, and Gemini.

## What Your Agent Gets

With Vibecode CLI, your agent can work with:

- Secure sandboxed runtime infrastructure
- 30+ integrations and APIs
- Built-in databases and authentication
- Built-in backends
- Deployments to custom domains or the App Store

## How It Works

1. Install the CLI by following the README in the Vibecode CLI repository
2. Sign in to get your API key or setup instructions
3. Paste the generated instructions into your agent
4. Let the agent build against Vibecode infrastructure

## Compatible Agents

Vibecode CLI is positioned as compatible with major agent tools, including:

- Cursor
- Claude
- Codex
- Gemini
