Resend is a modern email API service designed for developers to send transactional and marketing emails.

It provides reliable email delivery, beautiful templates, analytics, and a developer-friendly API for building email functionality into applications.

## Features

- Transactional email
- Marketing email support
- Email templates
- Delivery analytics
- Developer-friendly API
- Reliable delivery

## Use Cases

- Transactional emails
- User notifications
- Password resets
- Marketing campaigns
- Email verification
