Ideogram 3.0 is a great image generation model with a unique superpower: character references.

With just a single image, you can generate new images featuring that same character in different scenes and situations. Perfect for creating consistent characters across your app.

## Features

- Character reference system
- Consistent character generation
- Multiple scene generation
- High-quality output
- Creative flexibility

## Use Cases

- Character design consistency
- Story illustration
- Comic creation
- Game asset generation
- Character-driven content
